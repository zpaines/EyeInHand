Detailed Description

Driver class for the REMS Eye in Hand Registration Package.

This class handles all user interface and farms out functionality to other classes. It was designed to be fairly modular and easily extendible s.t. more menu options and functionality could be added without major code changes.

Member Function Documentation

Interface::MenuOption Interface::getCommand ( )

This gets called to display the menu to the user and handle the latest command. As well as returning the menu selection, the function also completes the user request. In other words, if the user selects to read a point cloud from a file, this function will read the point cloud from the file and then return the command choice (MenuOption::LoadFrame)
